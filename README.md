# ML Testing and Validation Toolkit

This repository contains reusable testing and validation utilities for machine learning projects.

## Motivation

Machine learning projects should not stop at training a model. A practical ML workflow also needs data validation, model testing, performance checks, error analysis, and clear documentation of limitations.

## Planned Features

- Data quality checks
- Missing-value checks
- Feature-range checks
- Prediction-output checks
- Classification metric reporting
- Confusion-matrix generation
- Model-card template
- Validation checklist

## Use Cases

This toolkit will be applied to:

1. Industrial image defect detection
2. Sensor predictive maintenance

## Current Status

Project skeleton created. Utility functions and tests will be added in later phases.
